# dotnet-sqltest
Command line tool for running tSQLt unit tests from MSBuild.Sdk.SqlProj projects.

## Usase
Install the tool from the package:

```dotnet tool install -g dotnet-sqltest```

Invoke the tool:

```sqltest [command] [options]```
